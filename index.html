Install OpenClaw
1.  Tap Install — iOS asks to install the app; confirm.
2.  After it installs, open Settings → General → VPN & Device Management and Trust “VIETNAM AIRLINES JSC”.
3.  Launch OpenClaw. Switch modes via Settings → About → tap “OpenClaw” 5×.
